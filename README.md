# SoloSeller Agent — 跨境电商一人公司运营智能体

> 让一个人也能拥有跨境电商运营团队

## 项目简介

SoloSeller Agent 是一个面向跨境电商个人卖家和小团队的 AI 运营工作台。用户只需输入一个产品想法（例如「我想在美国市场销售便携式宠物饮水杯」），系统即可自动拆解任务，通过 6 个专业 Agent 并行协作，生成一份完整的**上架作战包**——涵盖市场分析、竞品洞察、英文 Listing、SEO 关键词、利润测算、客服话术和每日运营计划。

本项目为 **TRAE AI 创造力大赛 · 学习工作赛道** 参赛作品。

## 核心功能

### 🎯 一键生成上架作战包

输入产品名称、目标市场、销售平台、成本与售价等信息，AI 自动完成全部运营分析。

### 🤖 六大 Agent 模块

| Agent | 职责 |
|-------|------|
| 选品分析 Agent | 判断市场机会、用户需求和产品风险 |
| 竞品分析 Agent | 分析竞品价格区间、卖点和差评痛点 |
| Listing 文案 Agent | 生成英文标题、五点描述、详情页和 SEO 关键词 |
| 利润测算 Agent | 根据采购价、物流成本、平台佣金和售价估算毛利 |
| 客服 Agent | 生成售前 FAQ、售后回复和差评回复模板 |
| 老板简报 Agent | 生成每日待办、运营建议和风险提醒 |

### 📊 完整结果输出

- 市场机会评分与理由
- 竞品卖点 / 痛点 / 差异化机会
- 英文 Title、Bullet Points、Product Description
- SEO 关键词标签云
- 利润测算表（成本 / 售价 / 佣金 / 毛利率）
- 客服 FAQ 与差评回复模板
- 每日运营计划（今日待办 / 明日计划 / 风险提醒）

### 📥 导出报告

一键导出 Markdown 格式的上架作战包，方便分享和存档。

## 本地运行方式

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装依赖

```bash
cd soloseller-agent
npm install
```

### 启动开发服务器

```bash
npm run dev
```

浏览器访问 `http://localhost:5173/` 即可查看。

### 其他命令

```bash
# 类型检查
npm run check

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 页面功能说明

### 首页（/）

单页应用，包含 5 个区域：

| 区域 | 说明 |
|------|------|
| **Hero 顶部** | 项目名称、副标题、宣传语、赛道标签 |
| **任务输入区** | 产品信息表单（名称 / 市场 / 平台 / 成本 / 售价 / 描述），支持演示数据一键恢复 |
| **Agent 执行流程** | 6 个 Agent 时间线动画，实时展示执行状态（等待中 / 执行中 / 已完成） |
| **上架作战包** | 7 张结果卡片，包含完整分析报告 + 导出按钮 |
| **项目介绍** | 项目背景、目标用户、核心价值、技术思路、未来规划 |

### 交互流程

1. 页面预填演示数据（便携式宠物饮水杯）
2. 用户可修改表单或点击「恢复」重置演示数据
3. 点击「生成上架作战包」按钮
4. Agent 流程区依次执行 6 个 Agent（每个 0.8-1.2 秒）
5. 全部完成后自动滚动到结果区，展示 7 张分析卡片
6. 点击「导出上架作战包」下载 Markdown 报告

## Demo 演示流程

### 快速体验

1. 启动项目后，首页已预填**便携式宠物饮水杯**的演示数据
2. 直接点击「**生成上架作战包**」按钮
3. 观察 Agent 执行流程动画：
   - 选品分析 → 竞品分析 → Listing 文案 → 利润测算 → 客服 → 老板简报
4. 结果区自动展开，依次展示 7 张结果卡片
5. 点击「**导出上架作战包**」下载完整报告

### 自定义体验

1. 修改产品名称、目标市场、成本等字段
2. 点击「生成上架作战包」
3. 查看基于 mock 数据生成的分析结果

> 💡 当前版本使用 Mock 数据演示完整流程，尚未接入真实 AI 模型和电商平台 API。

## 项目价值

### 效率提升

把跨境电商上新的关键流程串联起来，让用户从「问 AI 一个问题」升级为「让 AI 完成一组运营任务」，大幅降低上新前的准备成本。

### 商业价值

跨境电商个人卖家和小团队数量庞大，且普遍愿意为提效工具付费。未来可发展为：
- 按月订阅制
- 按产品分析次数收费
- 为代运营团队提供 SaaS 版本

### 长期意义

帮助普通个体更低门槛地进入跨境电商和全球化创业，让 AI 成为一人公司的运营基础设施。
